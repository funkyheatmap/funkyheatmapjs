# funkyheatmap(data, columnInfo, rowInfo, columnGroups, rowGroups, palettes, legends, positionArgs, options, scaleColumn) → {SVGElement}

The main entry point for the library. Takes data and various configuration options and returns an SVG element with the heatmap.

Parameters:
Name Type Description
data Object | Array.<Object>

data to plot, usually d3-fetch output. It should be an Array of Objects, each object has the same properties.
columnInfo Object | Array.<Object>

information about how the columns should be displayed
rowInfo Object | Array.<Object>

information about how the rows should be displayed
columnGroups Object | Array.<Object>

information about how to group columns
rowGroups Object | Array.<Object>

information about how to group rows
palettes Object

mapping of names to palette colors, see module:palettes.assignPalettes
legends Object | Array.<Object>

a list of legends to add to the plot
positionArgs Object

positioning arguments, see PositionArgs
options Object

options for the heatmap
fontSize int

font size for all text
scaleColumn boolean

whether to apply min-max scaling to numerical columns. Defaults to true
